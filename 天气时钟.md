# 硬件选型：

主控 STM32F103C8T6;

显示屏 T7735（SPI通信）; 

WIFI模块 ESP32C3 Supermini；

环境温度、加速度计：MPU6050；

调试串口: TTL转串口CH340；

# 设计方案：

1. 按图示显示信息内容
   1. 天气使用图标代替，当特殊天气没有图标时，显示文字内容，文字为黄色
   2. 温度用天蓝色显示，环境温度用浅绿色显示
   3. 时间用橘色显示，左上角显示年月日，中间显示时分秒
   4. 网络信息显示联网状态，当网络断开时，显示网络异常提示，正常状态显示白色，异常显示红色
   5. 背景黑色
2. 开机显示网络连接状态
3. 时间显示每100ms更新一次，时间每1h联网同步一次，防止时间误差
4. 天气每10min联网更新一次
5. 环境温度每10s更新一次
6. 网络信息每30s更新一次
7. 【屏幕自动旋转】
8. 【温度字体颜色随温度变化】
